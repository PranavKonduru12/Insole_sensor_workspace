# Insole_sensor_workspace
The purpose of this project is to create an embedded system that is capable of capturing and transmitting data that can be used to identify one's posture behavior. 

Embedded project programs the Adafruit nrf52840 sense microcontroller to samples readings from different sensors (FSR pressure sensors and IMU) into discrete digital values and transmits the data wirelessly through BLE.  

The programming is done on Nordic semiconductor's nRF Wireless Connect, which is the SDK used to control microcontrollers in the nRF family for your needs.
